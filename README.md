## About

Liberu Social Network is a fully open-source, modular social networking platform built on **Laravel 12**, **PHP 8.5**, **Filament 5**, and **Livewire 4**. It provides a production-ready foundation for building community-driven applications with modern tooling, a rich admin panel, and an extensible architecture.

Whether you are a developer looking to launch a niche social platform, an enterprise that needs a self-hosted community tool, or an open-source enthusiast, Liberu Social Network gives you a complete, battle-tested starting point you can own and customise.

Key highlights:
- 🚀 **Modern stack** — Laravel 12, PHP 8.5, Livewire 4, Filament 5
- 🔐 **Privacy-first** — granular privacy settings per user
- 🧩 **Modular architecture** — enable or disable features without touching core code
- ⚡ **High performance** — optional RoadRunner/Octane support
- 🛡️ **Role-based access control** — powered by Spatie Laravel Permission and Filament Shield
- 🌐 **REST API** — full API for friends, followers, messaging, and search

## Features

- **Authentication** — Jetstream with optional Socialstream (social login)
- **User Profiles** — avatars, bios, and customisable profile pages
- **Friend Request System**
  - Send, accept, and reject friend requests
  - View friends list and pending requests
  - Friend count displayed on profiles
- **Follower System**
  - Follow and unfollow other users
  - View followers and following lists
  - Follower/following counts on profiles
- **User Search** — search by name or email with quick add/follow actions
- **Private Messaging** — direct conversations between users
- **Real-time Notifications** — instant alerts for social interactions
- **Activity Feed** — chronological stream of network activity
- **Media Sharing** — upload and share images and other media
- **Groups** — create and manage community groups
- **Admin Panel** (Filament 5)
  - User management with role-based access control
  - Role and permission management via Filament Shield
  - Email verification management
  - Account status controls
  - Module management
  - Menu management
  - User statistics widgets
- **RESTful API**
  - Friend request endpoints (send, accept, reject)
  - Follower management endpoints
  - User search API
  - Full [API documentation](docs/FRIEND_FOLLOWER_API.md) available
- **Modular Architecture** — add or remove features via the module system

### Admin Panel

The admin panel is accessible at `/admin` and provides comprehensive management capabilities:

- **User Management**: Create, edit, and delete users with role assignment
- **Role & Permission System**: Powered by Spatie Laravel Permission and Filament Shield
  - Super Admin: Full system access
  - Admin: Standard administrative privileges
  - Panel User: Basic panel access
  - Free: Limited view-only permissions
- **Dashboard**: Overview of system statistics and quick actions
- **Module System**: Enable/disable application modules
- **Menu Management**: Configure navigation menus

For detailed documentation, see [Admin Panel User Management Guide](docs/ADMIN_PANEL_USER_MANAGEMENT.md).

## Installation

**Requirements:** PHP 8.5+, Composer, Node.js, and (optionally) Docker.

### Option 1 — Interactive command-line installer (recommended)

Run the interactive installer script from the project root:

```bash
bash setup.sh
```

The script will guide you through copying `.env.example` to `.env`, setting your database credentials, installing dependencies, running migrations, seeding the database, running tests, and optionally starting the development server. For users who prefer a graphical experience, a GUI-based setup wizard is also available — follow the on-screen prompts to complete each step.

### Option 2 — Manual setup

```bash
# 1. Copy environment file
cp .env.example .env

# 2. Install PHP dependencies
composer install

# 3. Generate application key
php artisan key:generate

# 4. Configure .env (DB credentials, mail, app URL), then run:
php artisan migrate --seed

# 5. Install and build frontend assets
npm install && npm run build

# 6. Start the development server
php artisan serve --host=127.0.0.1 --port=8000
```

### Option 3 — Docker

```bash
docker build -t liberu-social-network .
docker run -p 8000:8000 liberu-social-network
```

### Option 4 — Laravel Sail

```bash
./vendor/bin/sail up
```

> **Note:** Ensure your `.env` database credentials are correct before running migrations. The `setup.sh` script will prompt you to confirm your configuration before proceeding.
